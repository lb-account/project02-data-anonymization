# Differential Privacy: A Study on Privacy-Preserving Techniques  

## 📌 Overview  
This project explores differential privacy and related privacy-preserving techniques in data analysis and machine learning. It covers key concepts such as pseudonymization, membership inference attacks, k-anonymity, the Laplace mechanism, and the Private Aggregation of Teacher Ensembles (PATE) framework.  

## 📂 Contents  
- **Pseudonymization** – Basic data protection method and its limitations  
- **Membership Inference Attack** – A threat to privacy in machine learning models  
- **k-Anonymity** – A traditional approach to anonymization and its weaknesses  
- **Differential Privacy** – Theoretical foundation, sensitivity, and the Laplace mechanism  
- **PATE Framework** – A privacy-preserving approach using teacher-student models

- **numpy** – Numerical computing  
- **matplotlib** – Data visualization
